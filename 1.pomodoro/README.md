# 🍅 ポモドーロタイマー

Flask ベースの高機能ポモドーロタイマーアプリケーション

## 機能

### パターンA: 視覚的フィードバックの強化

- **円形プログレスバー**: 残り時間に応じて滑らかに減少するアニメーション
- **色の変化**: 時間経過に応じて青→黄→赤にグラデーション変化
- **背景エフェクト**: 集中時間中は背景にパーティクルエフェクトや波紋アニメーション

### パターンB: カスタマイズ性の向上

- **時間設定の柔軟化**: 25分固定ではなく、15/25/35/45分から選択可能
- **テーマ切り替え**: ダーク/ライト/フォーカスモード（ミニマル）
- **サウンド設定**: 開始音/終了音/tick音のオン/オフ切り替え
- **休憩時間カスタム**: 5/10/15分から選択

### パターンC: ゲーミフィケーション要素の追加

- **経験値システム**: 完了したポモドーロに応じてXPとレベルアップ（1ポモドーロ = 25 XP）
- **達成バッジ**: 「3日連続」「7日連続」「今週10回完了」「初回完了」「50回達成」などの実績システム
- **週間/月間統計**: より詳細なグラフ表示（完了率、平均集中時間など）
- **ストリーク表示**: 連続日数のカウント表示

## セットアップ

### 必要要件

- Python 3.8以上
- pip

### インストール

```bash
cd 1.pomodoro
pip install -r requirements.txt
```

### 実行

```bash
python app.py
```

ブラウザで `http://localhost:5000` にアクセスしてください。

## テスト

```bash
pytest tests/ -v
```

## 使用方法

1. **タイマーの開始**: 「開始」ボタンをクリックしてポモドーロを開始
2. **設定のカスタマイズ**: 「設定」タブで時間、テーマ、サウンドをカスタマイズ
3. **統計の確認**: 「統計」タブで週間・月間の完了状況を確認
4. **ゲーミフィケーション**: ポモドーロを完了してXP、レベル、バッジを獲得

## データ保存

- `user_data.json`: ユーザーの統計情報、XP、レベル、バッジ情報
- `settings.json`: ユーザーのカスタム設定

これらのファイルは自動的に生成され、`.gitignore`に含まれています。

## 技術スタック

- **バックエンド**: Flask 3.0.0
- **フロントエンド**: バニラJavaScript、HTML5、CSS3
- **テスト**: pytest、pytest-flask
- **ビジュアルエフェクト**: Canvas API (パーティクルアニメーション)
- **サウンド**: Web Audio API

## テスト目的

- **パターンA**: 視覚的な没入感がユーザーの集中力に与える影響を測定
- **パターンB**: 個人の好みに合わせた設定がユーザー継続率に与える影響を測定
- **パターンC**: ゲーミフィケーション要素がモチベーション維持と継続利用に与える影響を測定
